# New Oracle Groups Working Map

Дата фиксации: 2026-04-25

Этот документ нужен как рабочая карта перехода на новую архитектуру колоды.
Он не заменяет мастер-документ, а помогает работать по группам целиком.

Связанный мастер-документ:
- `docs/NEW_ORACLE_ARCHITECTURE_MASTER.md`

Статус:
- новая архитектура зафиксирована как рабочая
- текущий runtime еще не синхронизирован с ней полностью
- дальше работа идет по группам, а не по разрозненным картам

## Что уже решено

- `Хранитель Фонаря` уходит в `fate_path`
- `Тёмный Гость` заменяет старую функцию `Стража Ночи` в `family_circle`
- `Хранитель Нитей` остаётся мостовой картой
- `Хранительница Лунной Вуали` — финальное рабочее имя

## Как работаем дальше

1. Берём одну новую группу целиком.
2. Смотрим весь состав группы сразу.
3. Разводим карты по функциям внутри группы.
4. Переименовываем спорные карты.
5. Переписываем `Послание` и `Тень`.
6. Только после этого синхронизируем runtime и архитектурные таблицы.
