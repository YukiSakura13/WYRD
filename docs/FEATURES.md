# Features: WYRD

## Статусы

- `live` — уже реализовано
- `planned` — согласовано как следующее направление
- `idea` — идея без подтверждённого приоритета
- `tech-debt` — нужно улучшить существующее поведение

---

## Текущие функции

### Cover / вход

- `live` Стартовый cover-screen с входом в приложение
- `live` Вход без отдельной навигации и без роутинга

### Карта дня

- `live` Одна бесплатная карта в день
- `live` Проверка дневного лимита через локальное состояние
- `tech-debt` Логика границы суток зависит от локального времени браузера

### Чтение карты

- `live` Случайный выбор одной карты
- `live` Показ основного послания
- `live` Показ тени
- `live` Углубление значения через дополнительный сценарий
- `tech-debt` Углубление работает как фиксированный шаблон, а не как индивидуальный контент карты

### Расклады

- `live` Расклад на 3 карты
- `live` Расклад на 5 карт
- `tech-debt` Расклад не имеет собственной полноценной доменной модели
- `tech-debt` История раскладов не сохраняется

### Профиль и история

- `live` Локальный профиль
- `live` История одиночных чтений
- `tech-debt` История не включает расклады
- `tech-debt` Нет фильтрации и структуры истории

### Paywall / монетизация

- `live` Локальный paywall-preview для дополнительных действий
- `tech-debt` Это не реальный биллинг и не отдельная платёжная подсистема

### Аудио и атмосфера

- `live` Переключение звука
- `live` Синтезируемый фоновый лесной звук
- `live` Шуршание карт
- `tech-debt` Лежащий в проекте аудиофайл не используется

### PWA

- `live` Манифест приложения
- `live` Регистрация service worker
- `live` Базовый офлайн-кэш
- `tech-debt` Список кэшируемых файлов поддерживается вручную

---

## Запланированные функции

- `planned` Реальный экран ритуала с обратным отсчётом
- `planned` Полноценная модель расклада
- `planned` История раскладов
- `planned` Отделение контента от JS-логики
- `planned` Улучшение структуры профиля
- `planned` Более формализованная модель paywall/доступов

## Идеи

- `idea` Тематические режимы раскладов
- `idea` Индивидуальные тексты углубления для каждой карты
- `idea` Больше состояний атмосферы и звуковых режимов
- `idea` Экспорт истории чтений
- `idea` Мягкая персонализация профиля
