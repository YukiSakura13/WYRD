# Project Brief: WYRD

## Что это за проект

`WYRD` — атмосферный PWA-сайт/приложение в жанре мистического оракула духов леса. Пользователь получает символическое послание через карты, может углублять чтение и открывать расклады.

---

## Текущий формат продукта

- одностраничное веб-приложение
- работает без сборщика
- поддерживает офлайн-режим через PWA
- хранит пользовательское состояние локально
- построено вокруг эмоции, атмосферы, ритуальности и коротких символических интерпретаций

---

## Продуктовая цель

Сделать цельный атмосферный цифровой опыт, где:

- пользователь быстро входит в настроение
- получает понятный и выразительный ритуал взаимодействия
- возвращается ежедневно за новой картой
- постепенно вовлекается в более глубокие сценарии

---

## Основная ценность для пользователя

- короткий эмоциональный ритуал
- визуально цельный мистический опыт
- ежедневная точка контакта
- ощущение личного послания и символического смысла

---

## Текущее ядро сценариев

- вход через cover-screen
- бесплатная карта дня
- дополнительная карта
- углубление значения
- расклад на 3 карты
- расклад на 5 карт
- профиль и история

---

## Текущее состояние реализации

Сейчас проект уже имеет:

- рабочую модульную структуру
- централизованный store
- единый источник данных карт
- PWA-обвязку
- базовый звуковой слой

При этом остаются зоны роста:

- нет настоящего ритуального промежуточного экрана
- нет реального платёжного потока
- нет тестов
- контент пока жёстко пришит к JS
